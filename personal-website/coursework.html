Medical Image Analysis
Source Code
This was a very challenging course I took under Dr. Stephen M. Pizer. The course introduces various image modalities (X-Ray, MRI, CT etc), image preprocessing techniques (edges, bars, ridges, mathematical morphology etc), segmentation techniques and registration techniques. It's one of the toughest courses I've taken till date.
