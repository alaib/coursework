Benchmark datasets extracted from MHCBN databases.
The files are in weka arff and xrff formats. However, the files are also readable using any ASCII text editor.
For more information about these datasets, please see:
EL-Manzalawy Y, Dobbs D, Honavar V (2008) On evaluating MHC-II binding peptide prediction methods. PLoS ONE 3: e3268.
